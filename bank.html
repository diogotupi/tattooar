CAPAZ TATTOO · Banco AR

Para cada arte: envie a imagem impressa que a câmera deve reconhecer (boa luz, contraste) e o arquivo GLB da animação 3D que aparece em cima. A ordem das linhas define o índice do alvo (0, 1, 2…). Depois de compilar, os dados ficam salvos neste aparelho (IndexedDB).

A compilação roda no seu navegador (pode demorar). No celular use HTTPS (o comando npm run dev já ativa SSL local do Vite).
